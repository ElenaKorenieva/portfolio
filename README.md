## 🧠 Tech Stack

### Core
- React 19  
- TypeScript  
- Vite  

### Styling
- Tailwind CSS v4  
- Tailwind Vite Plugin  

### Animations & Interaction
- GSAP (ScrollTrigger, advanced UI animations)

### Routing
- React Router DOM  

### UI & Icons
- Lucide React  

### Tooling
- ESLint (modern configuration)  
- TypeScript ESLint  

### Deployment
- GitHub Pages  
- gh-pages
